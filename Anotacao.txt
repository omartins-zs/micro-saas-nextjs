Micro-SaaS na Prática: Setup com Next.js 14, Shadcn UI e Autenticação com Next Auth [+ Bônus]


Iniciando Projeto de Micro-Saas com NPM

npx create-next-app@14.1.0 micro-sass-nextjs



Instalando a Biblioteca Shadcn UI




npx shadcn-ui@latest init
npx shadcn-ui@latest add


npm install react-hook-form


npm install next-auth@beta
npm install prisma --save-dev

npx prisma init
